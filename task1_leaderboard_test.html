Task 1 Submission

  • STEP 1: Register on codalab

note: make sure to use the same e-mail address in the misp2021 and codalab, or you may not be allowed to submit the results in the competition. If you'd like to change e-mail address, please contact with organizers.

  • STEP 2: Apply to join in the competition and wait for approval.

tips: just check the box ( you don't need to download the term as you have signed it when you register in misp2021 competition ), and we'll permit access at that night.

  • STEP 3: Submit your result and check the scores

  • Make sure your submitted zip file as following directory structure:

    |summission.zip

    ​ ||--transcript.txt

  • Make sure every row of the transcript use the following format, which is a key-word separated by a space character:

segment_id 1or0

the segment id is named as follows:

< Room ID >_< Speakers IDs >_< Configuration ID >_< Index >_< Speaker ID >_< Start time >_< end time >

  • Here is a submission example and the whole segment_id list for development set provided. Note the number of segments in truth ground is smaller than those directly extracted from the transcript(.textgrid files), since some of the segments labeled in .textgrid files lack relevant signal or video segments in wav or mp4 files for early shutdown of recording device in the recording process.

  • When the status turns to "Finished" you can refresh the page the check the scores below. You can also click the button named “View scoring output log” to check if you miss any segments in your submission, for example:

stdout.txt

In your submission, some segments are missing. Please check and submit again!
Miss segments:
R05_S144145_C0001_I1_144_290733_291654
R05_S144145_C0001_I1_144_714266_715085
R05_S144145_C0001_I1_144_720176_721666

  • Other information

  • Make sure a. your submitted transcript, includes all segments in the development or evaluation set. b. your submission file format, or the submission will fail.

  • For each submitted score, it will be directly on the leader board if it's your best score so far.

  • During the evaluation set phase, every group is allowed to 3 times per day. For the development set phase, there is no limit.

  • If you meet any problems during submission, please contact with misp2021 administrators.
